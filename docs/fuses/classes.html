Input

Inputs are how Operators accept Parameters. Inputs are created via helper methods.

boolean Link:IsAnimated()

Returns true if any external source (connection, expression, etc) is driving this Input.

Output

Outputs give Parameters to other Operator's Inputs. Outputs are created via helper methods.

Operator

Objects that exist in the flow of Parameters. They have Inputs and Outputs. Operators include Tools and Modifiers.
