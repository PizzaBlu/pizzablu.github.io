Fusion Tags

Common Fusion Tags for Fuses & Macros

Link

General settings for Inputs & Outputs

TagTypeDescription
LINKID_DataTypeStringInputs & Outputs can only connect when they share the same DataType.
LINKS_NameStringThe name of this connection.
LINK_MainNumberInputs/Outputs with LINK_Main set to a positive integer show up on the node in the flow view. The first 4 Main input's have a special color assigned to them.

Input

General settings for Inputs.

TagTypeDescription
INP_ExternalBoolExternal Inputs can be connected to an external source for Animation and Expressions.
INP_PassiveBoolChanges to this input don't change the output of this tool.
INP_InteractivePassiveBoolLike INP_Passive but keeps Undo events.
INP_AddModifierStringClass ID of a CT_Modifier to create connected to this Input.
INP_DoNotifyChangedBoolChanges to this input fire the NotifyChanged event.
INP_InitialNotifyBoolAre NotifyChanged events fired when the tool is first added to the flow.

Number

General tags for Inputs with DataType "Number".

TagTypeDescription
INP_DefaultNumberDefault value for Number Inputs.
INP_MinAllowedNumberDo not allow this Number to be set below this value.
INP_MaxAllowedNumberDo not allow this Number to be set above this value.
INP_MinScaleNumberThe highest value on a slider.
INP_MaxScaleNumberThe lowest value on a slider.
INP_IntegerBoolMust be integer values only.
INP_SplineTypeStringWhat type of easing is used when interpolating between keyframes.

Point

General tags for Inputs with DataType "Point"

TagTypeDescription
INP_DefaultXNumberDefault X value for Point Inputs.
INP_DefaultYNumberDefault Y value for Point Inputs.

FuID

General tags for Inputs with DataType "FuID"

TagTypeDescription
INPID_DefaultIDStringDefault value for FuID Inputs.

Text

General tags for Inputs with DataType "Text"

TagTypeDescription
INPS_DefaultTextStringDefault value for Text Inputs.

InputControl

General tags for Inputs in the Inspector window

TagTypeDescription
INPID_InputControlStringThe Class ID of an InputControl to use in the Inspector window.
IC_NoResetBoolDisable the keyframe column for this control.
IC_NoLabelBoolDisable the label column for this control.
ICD_WidthNumberA width of 1 fills an entire row. Multiple controls can share a row when their widths add up to 1.
IC_ControlGroupNumberInputControls with the same ControlGroup number are in the same group.
IC_ControlIDNumberControlID specifies which Input this is within the ControlGroup.
IC_VisibleBoolHides the Input from the Inspector window.
IC_DisabledBoolDisables controls from the Inspector window.
IC_DisplayedPrecisionNumberHow many digits are shown after the decimal point in the Inspector.

LabelControl

An InputControl for adding extra info to the Inspector window. LabelControl can also be used to create Control Nests using the self:BeginControlNest() and self:EndControlNest() methods in Fuses.

TagTypeDescription
LBLC_MultiLineBoolThe hight of this LabelControl adapts to fit the content. <br> starts a new line.
LBLC_DropDownButtonBoolHides/Shows the Dropdown Arrow for NestControls.
LBLC_NumInputsNumberHow many inputs are Hidden with this Nest.
<br>HTMLLine break.
<h1,h2,h3,h4,h5>HTMLHeaders.
<font color=#ffffff>HTMLFont color.

SliderControl

An InputControl for Slider Inputs.

TagTypeDescription
SLCS_LowNameStringAn extra label below the left side of the slider.
SLCS_HighNameStringAn extra label below the right side of the slider.
SLCS_MidNameStringAn extra label below the middle of the slider.

ScrewControl

An InputControl for Screw Inputs.

OffsetControl

An InputControl for Point Inputs.

ComboControl

An InputControl for DataType Number dropdown Inputs. Holds the item number selected.

TagTypeDescription
CCS_AddStringString+Add a new entry to the dropdown.
CCS_AddSeparatorString+Add a separator to the dropdown, value is ignored.

ComboIDControl

An InputControl for DataType FuID dropdown Inputs.

TagTypeDescription
CCS_AddStringString+Add a new entry to the dropdown.
CCID_AddIDFuID+Insert after CCS_AddString to set the FuID value it returns when selected.
CCS_AddSeparatorString+Add a separator to the dropdown, value is ignored.

PreviewControl

General tags for Inputs in the viewer

TagTypeDescription
INPID_PreviewControlStringThe Class ID of a PreviewControl to use in the Viewer.
PC_VisibleBoolHides the PreviewControl from the viewer.

CrosshairControl

A PreviewControl for Point Inputs.

TagTypeDescription
CHC_StyleFuIDNormalCross, DiagonalCross, Rectangle, Circle, Rays

RectangleControl

A PreviewControl for Rectangles. Connects to multiple inputs, but lives on the Width Input.

TagTypeDescription
RCD_LockAspectNumberUse a fixed aspect ratio instead of a Height Input.
RCID_CenterStringID of an Input on this Node to be used as the center position.
RCID_AngleStringID of an Input on this Node to be used for the angle of the rectangle.
