Link

General tags for Inputs & Outputs

TagTypeDescription
LINKID_DataTypeStringInputs & Outputs can only connect when they share the same DataType.
LINKS_NameStringThe name of this Link. Also used for labels in the Inspector.
LINKID_IDStringThe ID of this Link. Also its key in the UserControls object in macros. Should not contain spaces.
LINK_MainNumberInputs/Outputs with LINK_Main set to a positive integer show up as Inputs/Outputs on the Node in the flow view. The first 4 Main input's have a special color assigned to them. LINK_Main also works on non-Image outputs!
LINK_NotColoredBoolOverride the Default Yellow,Green etc. colors for LINK_Main Inputs.

Example Image Link (valid as an Input or Output).

{
    LINKID_DataType = "Image",
    LINK_Main = 1,
}

InputControl

General tags for Inputs in the Inspector window

TagTypeDescription
INPID_InputControlStringThe Class ID of an InputControl to use in the Inspector window.
IC_NoResetBoolDisable the keyframe column for this control.
IC_NoLabelBoolDisable the label column for this control.
ICD_WidthNumberA width of 1 fills an entire row. Multiple controls can share a row when their widths add up to 1.
ICD_CenterNumberIf a center position is defined a slider will have different scales to the left and right of it, depending on INP_MinScale and INP_MaxScale.
IC_ControlGroupNumberInputControls with the same ControlGroup number are in the same group.
IC_ControlIDNumberControlID specifies which Input this is within the ControlGroup.
IC_VisibleBoolHides the Input from the Inspector window.
IC_DisabledBoolDisables controls from the Inspector window.
IC_DisplayedPrecisionNumberHow many digits are shown after the decimal point in the Inspector.
{
    LINKID_DataType = "Number",
    INPID_InputControl = "SliderControl",
}

LabelControl

An InputControl for adding extra info to the Inspector window. LabelControl can also be used to create Control Nests using the self:BeginControlNest() and self:EndControlNest() methods in Fuses. LabelControl can be used with any DataType.

TagTypeDescription
LBLC_MultiLineBoolThe height of this LabelControl adapts to fit the content. <br> starts a new line.
LBLC_DropDownButtonBoolHides/Shows the Dropdown Arrow for NestControls.
LBLC_NumInputsNumberHow many inputs are Hidden with this Nest.
<br>HTMLLine break in LINKS_Name.
<b>HTMLBold text in LINKS_Name.
<i>HTMLItalic text in LINKS_Name.
<h1,h2,h3,h4,h5>HTMLHeaders in LINKS_Name.
<font color=#ffffff>HTMLFont color in LINKS_Name.
{
    LINKS_Name = "<b>My Bold Title</b>",
    LINKID_DataType = "Number",
    INPID_InputControl = "LabelControl",
    INP_External = false,
    INP_Passive = true,
}

Examples

Adding a UserControls slider to a Macro setting.

UserControls = ordered() {
    MyControl = {
        LINKS_Name = "MyControl",
        LINKID_DataType = "Number",
        INPID_InputControl = "SliderControl",
        INP_Default = 0,
        INP_MinScale = 0,
        INP_MaxScale = 1,
    }
}

Adding a slider control to a Fuse.

InMyControl = self:AddInput("MyControl", "MyControl", {
    LINKID_DataType = "Number",
    INPID_InputControl = "SliderControl",
    INP_Default = 0,
    INP_MinScale = 0,
    INP_MaxScale = 1,
})

Transform controls in a fuse.


InCenter = self:AddInput("Center", "Center", {
    LINKID_DataType = "Point",
    INPID_InputControl = "OffsetControl",
    INPID_PreviewControl = "CrosshairControl",
})

InSize = self:AddInput("Size", "Size", {
    LINKID_DataType = "Number",

    INPID_InputControl = "SliderControl",
    INP_Default = 1.0,
    INP_MaxScale = 5.0,
    ICD_Center = 1.0,

    INPID_PreviewControl = "RectangleControl",
    RCID_Center = "Center",
    RCID_Angle = "Angle",
    RCD_LockAspect = 1.0,
})

InAngle = self:AddInput("Angle", "Angle", {
    LINKID_DataType = "Number",

    INPID_InputControl = "ScrewControl",
    INP_MaxScale = 360,

    INPID_PreviewControl = "AngleControl",
    ACID_Center = "Center",
})

Registry

General tags for FuRegisterClass

TagTypeDescription
REGS_NameStringNode name
REGS_URLStringDocumentation / help URL
REGS_VersionStringStringUsually a build date.
REG_VersionNumberVersion number.
REGS_OpIconStringStringString used in Toolbars (short).
REGS_CompanyStringCompany name used in about page.
REG_TimeVariantBoolThe output of the tool changes with the frame number.
REG_UnpredictableBoolAvoid caching this tool.
REGB_TemporalBoolSynonym for REG_TimeVariant.
REG_NoCommonCtrlsBoolDisable all common controls (such as Blend).
REG_OpNoMaskBoolDisable the mask Input.
REG_NoPreCalcProcessBoolUse the main Process() function instead of PreCalcProcess().
